# Welcome!
This is an unofficial mod of "Divagirls: Divas on Ice" for the Wii that was made as a replacement for the deleted "SuperSkating N****** for Wii" mod, developed entirely by Theodore Hung.

**Created Using The Following Tools:**
- BrawlCrate (https://github.com/soopercool101/BrawlCrate)
- WiiScrubber (https://forum.wii-homebrew.com/filebase/index.php/Entry/1328-WiiScrubber?__cf_chl_tk=ejgnSNM6ntTOosZsR39NMuryp1.nJnK3PnLjOUuY3XE-1777408113-1.0.1.1-xfSKbtphXlfOg2jqEUQ2UAPbF.rHW_XfeNDXtjjzTSU)
- Wii Partition Builder (included with WiiScrubber 1.4)

# System/Hardware Requirements

**For Users Playing on a Real Wii:**

Please note that depending on your Wii model, you may need a modchip, ODE, or homebrewed Wii in order to run the mod.
